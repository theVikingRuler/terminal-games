# Connect 4 (4 in a Row)
Connect 4, also known as "4 in a Row" is a turn-based 2 player game played commonly in a 6x7 board. Each player chooses to place their ball (marker) in one of the 7 columns. The ball goes to the last unoccupied box in that column. Each player's objective is to stack balls in a way to form a sequence of 4 in row, column, or diagonal.
## Installation
Make sure you have Python installed on your device.
1) Download the file "connect4.py" and save it in your Downloads folder.
2) Launch your Terminal, navigate to Downloads directory, and type "python3 connect4.py"
3) Take turns and enjoy the game.
